# Portfolio Victor Rubia - Développeur Full Stack

Portfolio style **VS Code / Terminal Workspace** - Design pour développeur.

## 🎨 Design Direction

**Signature** : "Code Editor Workspace"
- Layout inspiré VS Code
- Terminal prompt header
- File tree navigation
- Syntax highlighting colors
- Dark theme professionnel

**Palette** : VS Code Dark Theme
- Editor background: `#1E1E1E`
- Syntax colors: Keywords, variables, strings
- Terminal accents: Green, blue, orange
- Monospace: JetBrains Mono

---

## 🚀 Utilisation

**Ouvre `index.html` dans ton navigateur** - aucune installation nécessaire !

---

## ✏️ Modifier les Projets

Édite `script.js` → array `projects` :

```javascript
{
  title: "Nom projet",
  description: "Description...",
  type: "Full Stack",
  github: "https://github.com/...",
  demo: "https://...",
  tech: ["React", "Django"]
}
```

---

## 🛠️ Ta Stack

- React, Next.js, TypeScript
- Django, Python, Node.js
- Tailwind CSS, Bootstrap

---

## 🌐 Déploiement

**GitHub Pages** : Settings > Pages > Deploy from main
**Netlify** : Drag & drop sur netlify.com/drop
**Vercel** : `vercel --prod`

---

## 💡 Personnalisation

- **Couleurs** : `styles.css` → `:root`
- **Prompt** : Change `victor@dev` dans HTML
- **Stack badges** : Ajoute dans sidebar
- **Projets** : Modifie `script.js`

**Portfolio créé avec Craft - Design unique pour développeur !**
